# Signal Fusion with Meta-Learners: An explainable framework for data-driven decision support  

## Master's Thesis Project

This repository contains the implementation of a master's thesis focused on developing a data-driven decision-support framework using machine learning and signal fusion techniques.

The project investigates how information from multiple data sources can be combined into meaningful signals and integrated through a meta-learning model to support prioritization and decision-making in complex environments.

The framework emphasizes transparency, modularity, and practical applicability, aiming to support human judgment rather than replace it.
